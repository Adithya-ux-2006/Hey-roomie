# 🏠 Hey Nomads

Hey Nomads is a full-stack roommate matchmaking platform designed to help users find compatible living partners based on practical, real-world factors rather than superficial matching.

---

## 📌 Overview

Finding the right roommate is a decision that impacts daily life. Most platforms focus on listings or basic filters, but Hey Nomads takes a more structured approach by emphasizing compatibility.

The platform enables users to evaluate potential roommates based on lifestyle, budget, location, and move-in timelines, making the decision process more informed and reliable.

---

## 🚀 Key Features

* **Profile System**
  Users can create detailed profiles including lifestyle habits, preferences, and housing requirements.

* **Smart Matching**
  Matches are generated based on compatibility factors such as:

  * Lifestyle (sleep schedule, cleanliness, habits)
  * Budget alignment
  * Location preferences
  * Move-in timelines

* **Shortlist & Compare**
  Users can save potential matches and compare profiles side-by-side for better decision-making.

* **Roommate Fit Analysis**
  Provides insights into compatibility and highlights potential conflicts.

* **Chat System**
  Enables direct communication between matched users.

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone <your-repo-url>
cd hey-nomads
```

### 2. Run Backend

```bash
cd backend
npm install
npm run dev
```

### 3. Run Frontend

```bash
cd frontend-react
npm install
npm run dev
```

### 4. Open in browser

```
http://localhost:5173
```
